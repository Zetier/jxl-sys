# jxl-sys

Rust bindings to [libjxl](https://github.com/libjxl/libjxl)

## Building

Before building, you'll need to make sure that git submodules are initialized:
```bash
git submodule update --init --recursive
```

Then you can build with `cargo`:
```bash
cargo build
```
